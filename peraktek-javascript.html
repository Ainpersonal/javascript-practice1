05 — Loop for

Loop for — Ulangi sekian kali

Loop for dipakai saat kamu sudah tahu berapa kali ingin mengulang. Bayangkan kamu menghitung langkah dari 1 sampai 10 — kamu tahu angka awal dan akhirnya.

Anatomi loop for — 3 bagian dipisah titik koma:
for ( let i = 0 ; i < 5 ; i++ ) { ... }
① Mulai
set nilai awal i
② Syarat
lanjut jika masih true
③ Langkah
i++ = tambah 1 tiap putaran
Kode:
Output:
Contoh: hitung mundur
for (let i = 5; i >= 1; i--) {
  console.log(i);
}
// Output: 5, 4, 3, 2, 1
// i-- artinya kurangi 1 tiap putaran
Contoh: loop array buah
const buah = ["Apel", "Mangga", "Jeruk"];

for (let i = 0; i < buah.length; i++) {
  console.log(buah[i]);
}
// buah.length = 3
// Output: Apel, Mangga, Jeruk
06 — Loop while

Loop while — Ulangi selama syarat terpenuhi

Loop while dipakai saat kamu tidak tahu pasti berapa kali akan mengulang. Seperti menunggu sampai hujan berhenti — kamu tidak tahu akan berapa lama, hanya tahu kapan harus berhenti.

Perbandingan for vs while:
for — jumlah sudah diketahui
for (let i = 0; i < 5; i++) {
  // diulang tepat 5 kali
}
while — berhenti saat syarat false
let uang = 100;
while (uang > 0) {
  uang -= 25; // beli jajan
}
// berhenti saat uang habis
Kode:
Output:
Contoh: do...while
// do...while: jalankan DULU, cek syarat kemudian
// Dijamin jalan minimal 1 kali
let angka = 0;

do {
  console.log("Angka:", angka);
  angka++;
} while (angka < 3);

// Output: Angka: 0, Angka: 1, Angka: 2
Hati-hati: infinite loop!
// ❌ JANGAN dilakukan — loop tanpa akhir!
let i = 0;
while (true) {
  console.log(i);
  // lupa i++ → browser akan freeze!
}

// ✅ Selalu pastikan ada yang mengubah
// kondisi agar loop bisa berhenti
07 — break

break — Keluar paksa dari loop

break adalah tombol darurat — langsung menghentikan loop sepenuhnya meski syarat loop masih terpenuhi. Sangat berguna saat sudah menemukan data yang dicari.

Putaran 1 — cek... belum ketemu
Putaran 2 — cek... belum ketemu
Putaran 3 — KETEMU! → break 🛑
Putaran 4 — tidak dijalankan
Putaran 5 — tidak dijalankan
Kode:
Output:
Contoh: cari nama di daftar
const daftar = ["Ali", "Budi", "Citra", "Deni"];
const cari = "Citra";

for (let i = 0; i < daftar.length; i++) {
  if (daftar[i] === cari) {
    console.log("Ketemu di index:" + i);
    break; // stop! tidak perlu lanjut
  }
}
// Output: Ketemu di index: 2
08 — continue

continue — Lewati putaran ini saja

continue berbeda dengan break — ia tidak menghentikan loop, hanya melewati sisa kode di putaran ini dan langsung lanjut ke putaran berikutnya.

break → keluar total dari loop
1 ✓
2 ✓
3 🛑
4
5
continue → lewati putaran 3, tetap lanjut
1 ✓
2 ✓
3 ⏭
4 ✓
5 ✓
Kode:
Output:
Contoh: cetak angka ganjil saja
for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    continue; // lewati angka genap
  }
  console.log(i);
}
// Output: 1, 3, 5, 7, 9
Contoh: skip data kosong
const nilai = [80, null, 90, null, 75];

for (let i = 0; i < nilai.length; i++) {
  if (nilai[i] === null) continue;
  console.log("Nilai:", nilai[i]);
}
// Output: Nilai: 80, Nilai: 90, Nilai: 75
09 — Control Flow

Control Flow — Menentukan jalur program

Control flow adalah cara program memilih jalur mana yang dijalankan berdasarkan kondisi. Dengan if, else if, else, dan switch program bisa mengambil keputusan seperti manusia.

Simulator nilai rapor — geser angka dan lihat hasilnya
0 100 75 
const nilai = 75;

if (nilai >= 90) {
  grade = "A — Sempurna!"
} else if (nilai >= 80) {
  grade = "B — Bagus!"
} else if (nilai >= 70) {
  grade = "C — Cukup"
} else if (nilai >= 60) {
  grade = "D — Perlu usaha"
} else {
  grade = "E — Remedial"
}

Switch — lebih rapi untuk banyak kondisi spesifik

Kalau kondisinya banyak dan spesifik, switch lebih rapi dari if/else berantai.

if/else — makin banyak makin panjang
if (hari === "Senin") {
  console.log("Olahraga");
} else if (hari === "Rabu") {
  console.log("Matematika");
} else if (hari === "Jumat") {
  console.log("Pramuka");
} else {
  console.log("Bebas");
}
switch — sama hasilnya, lebih rapi
switch (hari) {
  case "Senin":
    console.log("Olahraga");
    break;
  case "Rabu":
    console.log("Matematika");
    break;
  case "Jumat":
    console.log("Pramuka");
    break;
  default:
    console.log("Bebas");
}
